# Alibaba Cloud DNS (International)

- **Provider id:** `aliyun_intl`
- **Endpoint:** `https://alidns.ap-southeast-1.aliyuncs.com`
- **Auth:** RPC HMAC-SHA1 (`common/aliyun_rpc.rs`)
- **Actions:** same as China, different endpoint
- **Separate from:** `aliyun_cn`

Docs: https://www.alibabacloud.com/help/en/dns/developer-reference/
