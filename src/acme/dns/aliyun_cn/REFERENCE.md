# Alibaba Cloud DNS (China)

- **Provider id:** `aliyun_cn`
- **Endpoint:** `https://alidns.aliyuncs.com`
- **Auth:** RPC HMAC-SHA1 (`common/aliyun_rpc.rs`)
- **Actions:** `AddDomainRecord`, `DescribeDomainRecords`, `DeleteDomainRecord`
- **Separate from:** `aliyun_intl` (different endpoint/keys)

Docs: https://www.alibabacloud.com/help/en/dns/developer-reference/
